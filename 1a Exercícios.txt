PROGRAMAS OBRIGATÓRIOS

A) Faça um programa que peça um número inteiro e imprima na tela se este número é positivo ou negativo.

B) Faça um programa que peça um número inteiro e imprima na tela se este número é par, ímpar ou zero.

C) Faça um programa que pergunte em que turno você estuda. 
   Peça para digitar M-matutino, V-vespertino ou N-noturno. 
   Imprima a mensagem “Bom Dia!”, “Boa Tarde!” ou “Boa Noite!” conforme o turno.
   Use o componente Ckeckbox.

D) Faça um programa que tendo como dados de entrada a altura e o sexo de uma pessoa, calcule o peso ideal, utilizando as seguintes fórmulas:
 •	Para homem: (72,7 * altura) – 58
 •	Para mulher: (62,1 * altura) – 44,7

E) Faça um programa  que verifique se uma letra digitada é vogal ou consoante ou um caracter desconhecido.
